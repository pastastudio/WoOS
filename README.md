## About

A demo website built with **Next.js**, exploring:

- Reducing unnecessary device purchases
- Comparing operating systems with a focus on efficiency
- Techniques for building performant, resource-aware software

### Background

This project was created as part of a university course. We weren't able to finish it in time for the original deadline, and picked it back up about six months later to keep working on it.
